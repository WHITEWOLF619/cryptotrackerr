# cryptotrackerr
A flutter based app that tracks real-time crypto prices in different currencies.

Cryptos-
   CNY
   EUR
   INR
   JPY
   USD

Currencies-
   BTC
   ETH
   ADA
   XMR
   BCH
   MATIC


using coinApi.Io

# Dependencies
flutter v- 1.21
Dart v- 1.21
Android studio v- 4.2
Android v- 10(Q)
